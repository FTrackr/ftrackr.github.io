Here you can set what information do you want to be displayed in the set details. Set details can be viewed by expanding it using the arrow found on the right side.

Show previous workout

Shows information about the same set from the previous workout: reps done, weight lifted, if it was completed, and other.

Show total added weight

Shows the total added weight, if it is equipped on two sides separately.

Show base resistance

Shows the base resistance without any added weight. If there is no base resistance in that exercise, it is not shown.

Show total weight

Shows the total weight lifted per rep, counting all sides (if applicable) and the base resistance (if applicable). If there is no base resistance in that exercise, it is not shown.

Show total weight in set

Shows the total weight lifted in that set, counting all sides (if applicable) and the base resistance (if applicable).

Show total weight in exercise

Shows the total weight lifted in that exercise up to this set, counting all sides (if applicable) and the base resistance (if applicable).

Show body mass percent

Shows the percent of body mass you lift in one rep, counting all sides (if applicable) and the base resistance (if applicable).

Show weight calculator

Shows the weight calculator that helps you put on the correct weights. There is a version of the calculator for exercises using plate weights as well as cable weights.

The weights you want to use in the calculator can be chosen from the "Available weights" setting!
